## üìå Project Overview

The Iris dataset contains measurements of iris flowers belonging to three species:
- **Setosa**
- **Versicolor**
- **Virginica**

Using supervised machine learning techniques, this project classifies flowers into one of these species with high accuracy.

---

## üß† Machine Learning Workflow

1. **Data Loading**
   - Iris dataset (locally downloaded)

2. **Exploratory Data Analysis (EDA)**
   - Feature distributions
   - Pair plots for feature relationships

3. **Data Preprocessing**
   - Feature scaling using `StandardScaler`
   - Train-test split

4. **Model Training**
   - Logistic Regression classifier
   - Model evaluation using accuracy, precision, recall, and F1-score

5. **Model Saving**
   - Trained model and scaler saved using `joblib`

6. **Prediction Interface**
   - Command Line Interface (CLI) for user input
   - Displays predicted species and confidence score
